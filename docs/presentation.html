Fonctionnalités

Ce que fait APSIA

💬

Chat IA & Import multimédia

Interface conversationnelle complète. Import PDF (50 Mo max) ou photo de cours manuscrit avec OCR automatique. Détection d'intention : APSIA comprend ce que l'élève veut faire et propose l'action correspondante.

🗺️

Circuits d'apprentissage

5 à 8 étapes pédagogiques structurées avec contenu détaillé (200–400 mots) et concepts-clés par étape. Cache SHA-256 pour éviter la re-génération.

🧪

Quiz adaptatifs

QCM à difficulté calibrée (facile / moyen / difficile), 5–20 questions. Ciblage des lacunes — les questions ratées génèrent automatiquement de nouvelles questions sur les mêmes concepts.

📝

Résumés condensés

Résumé 1-page par circuit : points essentiels par étape, glossaire 5–10 termes, conseils d'étude spécifiques au contenu.

💡

Assistant Chat IA

Conversation naturelle avec APSIA. Détection d'intention : répond aux questions, génère un circuit, lance un quiz ou produit un résumé selon le message.

🧠

Profil adaptatif

Onboarding 3 étapes (niveau, filière, style d'apprentissage). Analyse IA du profil → forces, faiblesses, conseils personnalisés. Dashboard mis à jour automatiquement après chaque quiz. Tag de personnalité unique pour chaque élève.

🏛️

Bibliothèque BEPC/BAC

400+ sujets d'examens guinéens classés par matière, filière et année (2010–2025). Corrigés intelligents générés par Claude. Circuit de révision associé à chaque sujet.

Logique de fonctionnement

Flux principal — De PDF à circuit

1
Sélection du PDF L'utilisateur sélectionne un fichier (max 50 Mo). Validation format + taille côté client.
2
Upload & chiffrement Upload vers Supabase Storage → chiffrement AES-256-GCM (Edge Function encrypt-document). Clé stockée dans Supabase Vault.
3
Analyse IA Edge Function analyze-document → Claude Opus lit le PDF chiffré et retourne titre, matière, niveau, concepts-clés, résumé.
4
Vérification cache Hash SHA-256 calculé. Si un circuit identique existe déjà (même document ou même hash) → retour immédiat sans appel IA.
5
Génération du circuit Edge Function generate-circuit → Claude Opus génère 5–8 étapes structurées et les persiste en base (tables circuits + circuit_steps).
✓
Redirection vers le circuit L'utilisateur atterrit sur /circuit/[id] et peut immédiatement commencer à apprendre.
Flux alternatif

Flux alternatif — Via Chat ou Photo

1
Photo de cours manuscrit OCR automatique → extraction du texte du cours à partir d'une photo prise depuis le téléphone.
2
OU — Message dans le Chat Détection d'intention par Claude Sonnet — APSIA comprend la demande et propose l'action appropriée.
3
OU — PDF importé depuis le Chat Même flux que le flux principal, déclenché directement depuis la conversation sans passer par l'écran d'upload.
4
Action proposée Selon le contenu détecté : génération de circuit · lancement de quiz · production de résumé · réponse directe.
5
Carte cliquable dans la conversation Le résultat s'affiche comme une carte interactive dans le chat — l'élève clique pour accéder directement au contenu généré.
Flux secondaires

Onboarding adaptatif

1
3 étapes questionnaire Niveau · filière · style d'apprentissage · temps disponible · matières · difficultés.
2
Analyse IA (timeout 20s) Claude Sonnet génère forces, faiblesses, plan d'étude personnalisé et message motivationnel.
3
Dashboard personnalisé Forces et faiblesses mis à jour après chaque quiz via Edge Function update-learning-profile.

Sécurité & Conformité

Protections en place

🔐

Chiffrement des données

Chaque PDF est chiffré individuellement en AES-256-GCM. La clé est stockée dans Supabase Vault (pgsodium), isolée par utilisateur.

🛡️

Authentification & RLS

Row Level Security sur toutes les tables. Tokens JWT vérifiés à chaque requête via Supabase GoTrue avec algorithme ES256 natif.

⏱️

Rate Limiting

3 fenêtres glissantes (minute / heure / jour) par opération et par plan. Bloc automatique avec message lisible et délai de retry.

📋

Audit Logs

Toutes les actions IA et admin sont tracées (user_id, IP, action, statut). Conformité RGPD avec purge automatique configurable.
